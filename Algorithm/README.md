### 시간복잡도
<details>
<summary></summary>

- 알고리즘 실행시 입력 데이터와 수행시간의 함수관계
- 효율적인 알고리즘을 나타내는 척도
- 입력 값이 커질수록 수행시간의 증가량 

</details>

### 공간복잡도
<details>
<summary></summary>

- 알고리즘 실행시 입력 데이터와 메모리 공간의 함수관계
- 효율적인 알고리즘을 나타내는 척도
- 입력 값이 커질수록 필요한 메모리 공간의 증가량 

</details>
